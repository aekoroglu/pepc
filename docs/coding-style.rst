Strings
=======

Whenever possible, use double quotes for strings. Do not use single quotes, unless you have to.
Examples.

*Do*:

* ``"my string"``
* ``mydict["element"]``
* ``if "a" in data:``

*Don't*:

* ``'my string'``
* ``mydict['element']``
* ``if 'a' in data:``

Possible exceptoins example:

* ``f"dictionary elemeht {mydict['element']}"``
